Histogram ảnh là biểu đồ biểu diễn sự phân bố mức xám (độ sáng) của các pixel trong ảnh.

Trục X: Giá trị mức xám từ 0 → 255.

0: màu đen hoàn toàn.

255: màu trắng hoàn toàn.

Trục Y: Số lượng pixel có cùng mức xám.

--------------------------------

Ý nghĩa

Histogram tập trung bên trái → ảnh tối.

Histogram tập trung bên phải → ảnh sáng.

Histogram trải rộng trên toàn dải 0–255 → ảnh có độ tương phản cao.

Histogram tập trung ở giữa → ảnh có độ tương phản thấp.

------------------------------------------



Ứng dụng

Phân tích độ sáng và độ tương phản của ảnh.

Cân bằng histogram (Histogram Equalization).

Phân ngưỡng ảnh (Thresholding).

Tiền xử lý trong nhận dạng và xử lý ảnh.
